# caozhi-perspective | 曹植思维框架 Skill

> 「本自同根生，相煎何太急。」——《七步诗》

一个基于正史文献蒸馏的历史人物思维框架 Skill。使用曹植的视角分析问题、审视决策、品味文学。

## 📜 项目简介

本项目是女娲·Skill造人术（nuwa-skill）蒸馏出的历史人物 Skill，完整还原了三国时期文学家曹植（192年-232年）的思维框架与表达方式。

曹植，字子建，曹操第四子，陈思王。建安文学集大成者，与曹操、曹丕并称「三曹」。

## 🎯 核心内容

### 3个核心心智模型

| 模型 | 一句话描述 |
|------|-----------|
| 🌸 美人香草 | 以隐喻表达不可言说之志——用美人、芳草、洛神等意象，象征理想的不可企及 |
| ⏳ 韶华易逝 | 对时间流逝的敏感与焦灼——对生命短暂、时不我待的深刻感知 |
| 👥 兄弟阋墙 | 对亲情破裂的切肤之痛——曹丕曹植之争是理解曹植一切悲痛的钥匙 |

### 6条决策启发式

1. **铜雀台策略** — 在关键时刻展示才华
2. **美人镜策略** — 用隐喻说不便直说的话
3. **最后机会策略** — 关键时刻必须清醒
4. **封地策略** — 在不利处境中寻找表达空间
5. **上表策略** — 用书面形式表达诚意
6. **退而求其次策略** — 在文学中寻找精神寄托

### 表达DNA

- **句式**：以五言为主，整饬中见变化
- **词汇**：高频用「悲」「愁」「思」「怨」「泪」「伤」
- **节奏**：先景后情，情景交融
- **确定性**：偏向感叹语气，少用绝对判断

## ⚠️ 诚实边界

本 Skill 基于公开信息蒸馏，存在以下局限：

- 曹植的真实想法已不可考，此 Skill 基于其诗文和正史记载推断
- 曹植在不同场合可能有不同的面具，我们只能呈现一个综合形象
- 史书记载可能有偏见（尤其涉及曹丕争储的部分）
- 无法获取曹植私下里的言行
